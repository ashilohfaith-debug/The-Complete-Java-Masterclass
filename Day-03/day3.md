08.05.2026

## Overview
Today, I learnt about Operators, Expressions, Assignment Operators, and Handling User Input in Java using the Scanner class.

<br/>

## Topics Covered
1️⃣ Handling User Input

Java uses the Scanner class from `java.util` package to take user input from the keyboard. <br/>
🔹 Syntax for importing Scanner class:
```
import java.util.Scanner;
```
🔹 Syntax for creating Scanner objects:
```
Scanner sc = new Scanner(System.in);
```

<br/>

2️⃣ Introduction to Operators

Operators are special symbols used to perform operations on variables and values.

<br/>

3️⃣ Expressions

Expressions are combinations of variables, operators, and values that produce a result.

<br/>

4️⃣ Arithmetic Operators

They are used for mathematical operations. They are as follows: <br/>
🔹 Addition (+) <br/>
🔹 Subtraction (-) <br/>
🔹 Multiplication (*) <br/>
🔹 Division (/) <br/>
🔹 Modulus (%)

<br/>

5️⃣ Unary Operators

Operate on a single operand.  They are as follows: <br/>
🔹 Unary plus (+) <br/>
🔹 Unary minus (-) <br/>
🔹 Increment (++) <br/>
🔹 Decrement (--) 

<br/>

6️⃣ Relational Operators

Used to compare values.  They are as follows: <br/>
🔹 Equal to (==) <br/>
🔹 Not equal to (!=) <br/>
🔹 Greater than (>) <br/>
🔹 Less than (<) <br/>
🔹 Greater than or equal to (>=) <br/>
🔹 Less than or equal to (<=)

<br/>

7️⃣ Logical Operators

Used to perform logical comparisons. They are as follows: <br/>
🔹 AND (&&) <br/>
🔹 OR (||) <br/>
🔹 NOT (!)

<br/>

8️⃣ Short-Circuit Logical Operators

In short-circuit evaluation, the second condition is checked only if necessary. Example:
```
boolean result = (a > 0) && (b < 15);
```

<br/>

9️⃣ Assignment Operators

Used to assign or update values. They are as follows: <br/>
🔹 a = 5; <br/>
🔹 a += 5; <br/>
🔹 a -= 5; <br/>
🔹 a *= 5; <br/>
🔹 a /= 5; <br/>
🔹 a %= 5;

<br/>

🔟 Operator Precedence

Operator precedence determines the order of evaluation. <br/>

<br/>

## Key Takeaways
💡 Operators are useful for:

🔹 Decision making <br/>

🔹 Updating values dynamically
